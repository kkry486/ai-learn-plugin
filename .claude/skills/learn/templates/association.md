# 关联内化模板

## 指令

将新学概念与知识库中已有的相关主题进行对比分析，帮助用户建立知识网络。

## 输入

【新学概念】：{{new_concept}}
【所属主题】：{{new_topic}}
【知识库中检索到的相关主题】：{{related_topics}}
（如果为空，跳过对比，直接告知"这是该领域的第一个知识节点"）

## 任务

对每个检索到的相关主题，进行四维对比：

1. **问题域**：它们解决的是同一个问题吗？还是同一问题的不同阶段？
2. **可迁移经验**：已学概念的哪些直觉/经验可以直接帮助理解新概念？
3. **差异点**：新概念的哪些特性是已学概念没有的？
4. **位置关系**：如果画一张知识地图，这两个概念分别在哪？什么关系？
