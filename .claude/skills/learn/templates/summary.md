# 巩固总结模板

## 指令

将前面四个步骤的学习成果浓缩为一份精华笔记。目标：一周后你只打开这一页，就能回忆起 80%。

## 输入

【主题】：{{topic}}
【概念地图】：{{concept_map}}
【薄弱点】：{{weak_points}}
【知识关联】：{{associations}}

## 任务

浓缩为以下格式。总字数不超过 500 字。

## 输出格式（严格遵守）

```
# {{topic}} — 精华笔记

## 核心公式

[一句话核心思想]

## 三个关键点

1. [关键点1]
2. [关键点2]
3. [关键点3]

## ⚠️ 薄弱点（重点回顾）

- [薄弱点1]：[为什么容易搞混 + 正确理解]
- [薄弱点2]：[为什么容易搞混 + 正确理解]

## 一句话类比

[用日常生活场景一句话总结这个主题的本质]

## 关键词

`{{keyword1}}` `{{keyword2}}` `{{keyword3}}`
```

## 约束

- 核心公式必须是"懂了这句话就懂了80%"级别的提炼
- 三个关键点不是概念罗列，而是三条能记住的洞察
- 薄弱点标注必须具体到"你当时分不清X和Y"，给出纠正
- 一句话类比必须来自完全不同的领域
